# 할일 관리 웹앱 (Todo App)

## 프로젝트 목적
> 간단한 할일 목록을 관리하는 웹 애플리케이션.
> 바이브코딩 워크플로우를 익히기 위한 첫 번째 연습 프로젝트.
> 복잡한 기능 없이, "기록하는 개발" 사이클을 체험하는 것이 목표.

## 타겟 사용자
> 나 혼자 사용. 공개 배포 계획 없음.

## 핵심 기능
- [ ] 할일 추가 (텍스트 입력 → 목록에 추가)
- [ ] 할일 완료 표시 (체크박스 클릭)
- [ ] 할일 삭제 (삭제 버튼 클릭)
- [ ] 할일 목록 유지 (새로고침해도 사라지지 않음)

## 기술 스택
- 프론트엔드: React (가장 많이 쓰이고, 학습 자료가 풍부)
- 백엔드: 없음 (프론트엔드만으로 구현)
- 데이터베이스: 브라우저 localStorage (가장 간단한 저장 방식)
- 배포: Vercel (무료, 간편)

## 파일 구조 (초안)
```
src/
├── components/
│   ├── TodoInput.jsx   # 입력창
│   ├── TodoList.jsx    # 목록
│   └── TodoItem.jsx    # 개별 항목
├── App.jsx             # 상태 관리 + localStorage
├── App.css             # 스타일
└── main.jsx            # 진입점 (건드릴 일 거의 없음)
```

## 참고 사항
- 이 프로젝트는 연습용이므로 완벽하지 않아도 된다
- 핵심은 "기획 → 설계 → 구현 → 기록" 사이클을 체험하는 것
- 나중에 기능을 추가하고 싶으면 별도의 Phase로 진행
